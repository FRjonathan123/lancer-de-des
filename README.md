# 🎲Simulation de lancer de Dés
Une simulation en python pour analyser la distribution statistique de la somme de deux dés.

## 📝Description
Ce projet simule 15 000 lancers de deux dés à 6 faces et calcule le pourcentage d'apparition de chaque résultat (de 2 à 12).
Il met en évidence que le chiffre **7** est le résultat le plus probable statistiquement.

## 🚀Comment l'utiliser
1. Assurez-vous d'avoir Python installé.
2. Téléchargez le fichier `main.py`.
3. Lancez le script `python main.py`

## 📊Résultats typiques
| Résultat | Probabilité |
| :--- | :--- |
| 7 | ~16.6% |
| 2 / 12 | ~2.7% |
